# WorkSafe - Sistema de Gestão de ASO

Sistema desenvolvido para gerenciamento de ASOs (Atestados de Saúde Ocupacional), colaboradores, exames e controle de vencimentos.

## Tecnologias

* Java
* PostgreSQL
* IntelliJ IDEA
* Git/GitHub

## Funcionalidades

* Cadastro de colaboradores
* Cadastro de cargos
* Controle de ASO periódico
* Cálculo automático de vencimento
* Controle de exames ocupacionais

## Equipe

* Cauã da Paz
* Cauã Ramos

## Como executar

1. Clonar o projeto
2. Abrir no IntelliJ
3. Configurar banco PostgreSQL
4. Executar aplicação
